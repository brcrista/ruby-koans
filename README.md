## Prerequisites

- [Ruby](https://www.ruby-lang.org/en/)
- [rbenv](https://github.com/rbenv/rbenv) (optional) will use the `.ruby-version` file in the root.

## Running the koans

```bash
cd koans
ruby path_to_enlightenment.rb
```
